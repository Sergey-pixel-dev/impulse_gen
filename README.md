# SignalGenerator

Низкочастотный генератор сигналов на STM32F446RET6 с графическим интерфейсом на дисплее и десктопным приложением для настройки.

## Функционал

- Генерация сигналов стандартных форм (синус, меандр, треугольник, пила)
- Генерация сигналов произвольной формы — задаются через графическое приложение на ПК
- Настраиваемые параметры: частота, амплитуда, смещение, точность
- Отображение текущих параметров на дисплее ST7789
- Сохранение всех настроек и произвольных форм сигналов в EEPROM

## Аппаратная часть

- MCU: STM32F446RET6
- Дисплей: ST7789 (SPI)
- Энкодер + кнопки для навигации по меню
- EEPROM для хранения настроек
- Плата спаяна вручную на макетной плате

## Стек

C, STM32F446, CMSIS/HAL, DAC, DMA, SPI (дисплей), I2C (EEPROM), таймеры

## Связанные проекты

- Графическое приложение для задания произвольных сигналов (десктоп)

## Сборка

Проект собирается в STM32CubeIDE.
